# Introduction
This is an extension for the Functional Parallel Typified Language with syntax highlighting and hints.

# Snippets
You can use snippets to write some constructions of code.
Just start typing "Scheme", "Fun", "Fun[]", "Data[]", "if->".

# Howers
You can find useful information about some functions.
Just hover with mouse.

# Highlighting
You can change your color theme.
Just press Ctrl+K Ctrl+T

# Running
You can run your code.
Just press Ctrl+Shift+P and type "Run current FPTL file" or simply press "Run (FPTL)" button on statusbar.

# Quick info
You can open quick info.
Just press Ctrl+Shift+P and type "Show quick FPTL info".
